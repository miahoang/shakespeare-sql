To use SPIDER in the framework, please download [the SPIDER dataset](https://yale-lily.github.io/spider), then unzip the database files into this folder. The structure needs to look like this:

```
shakespear-sql/
├── data/                    
│   └── SPIDER/                    
│       ├── spider_ambrosia_format.csv
│       ├── database/
│              ├── academic/
│              ├── activity_1/
│              ├── ...
│       ├── test_database/
│              ├── aan_1/
│              ├── academic/
│              ├── activity_1/
│              ├── ...
(the rest of the SPIDER files are not needed)
```
