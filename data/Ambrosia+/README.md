Respecting the wishes of original Ambrosia's authors, we refrained from uploading the dataset to Github.

Please download it [here](https://ambrosia-benchmark.github.io/), then unzip it into /shakespeare-sql/data folder so that the file structure looks like this:

```
shakespear-sql/
├── data/                    
│   └── Ambrosia+/                    
│       ├── ambrosia_with_unanswerable_validated.csv
│       ├── data/
│            ├── attachment/
│            ├── domains/
│            ├── scope/
│            ├── vague/
(the rest of the Ambrosia files are not needed)
```
